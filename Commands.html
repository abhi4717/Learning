1. To instruct UNIX to use vim instead of vi: Search for .vimrc file in the root. If present UNIX will use vim instead of vi. If not create one to use vim $ touch ~/.vimrc
Basic Commands
Mode Command
Insert i
To come back to command mode <ESC>
Navigate left h
Navigate down j
Navigate up k
Navigate right l
Delete Characters x
Undo changes u
Redo the changes ^R or Ctrl + R
Exit and save zz(did not work)
Quit discarding changes :q!
Appending characters after cursor position a
Deleting a line dd
Inserting a new line o
Getting help :help
Using count to repeat a command <Num><command to repeat>
eg. 3x
Editting Commands
Move forward between words w
Move backward between words b
Move end of line $
Move to first nonblank character in the line ^
Move to first character of the line 0
Search a character in right direction f<character to search>
Search a character in left direction F<Character to search>
Search till a character before in right direction t<Character to search>
Search till a character after in left direction T<Character to search>
Go to specific line G<Line number>
(Note: Without number G would got to last line in the file)
Show line numbers :set number
Remove line numbers :set nonumber
Show the line status Ctrl+G
Scroll Up Ctrl+U
Scroll Down Ctrl+D
Delete Line dd
Delete Word dw
Delete words <num>dw
or
d<num>
Repeat last delete or change command .
